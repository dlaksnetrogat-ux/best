# 🔑 Как получить YouTube API ключ

## ⚠️ ВАЖНО: Запуск локального сервера

YouTube не позволяет встраивать видео в файлы, открытые через `file://`

### Быстрый запуск (Windows):
1. Дважды кликни на файл `start-server.bat`
2. Откроется браузер с адресом http://localhost:8000
3. Готово! Теперь YouTube видео работают!

### Быстрый запуск (Mac/Linux):
```bash
./start-server.sh
```

### Ручной запуск:
```bash
# Python (если установлен)
python -m http.server 8000

# Или Node.js
npx http-server -p 8000
```

Затем открой: **http://localhost:8000**

## Шаг 1: Создать проект в Google Cloud Console

1. Перейди на https://console.cloud.google.com/
2. Войди в свой Google аккаунт
3. Нажми "Создать проект" (или выбери существующий)
4. Дай проекту имя, например "Mirtube"

## Шаг 2: Включить YouTube Data API v3

1. В меню слева выбери "API и сервисы" → "Библиотека"
2. Найди "YouTube Data API v3"
3. Нажми на него и нажми кнопку "Включить"

## Шаг 3: Создать API ключ

1. Перейди в "API и сервисы" → "Учетные данные"
2. Нажми "+ СОЗДАТЬ УЧЕТНЫЕ ДАННЫЕ"
3. Выбери "Ключ API"
4. Скопируй созданный ключ

## Шаг 4: Добавить ключ в Mirtube

1. Открой файл `script.js`
2. Найди строку:
   ```javascript
   const YOUTUBE_API_KEY = 'AIzaSyDemoKeyForTesting123456789';
   ```
3. Замени демо-ключ на свой настоящий ключ:
   ```javascript
   const YOUTUBE_API_KEY = 'ТВОЙ_НАСТОЯЩИЙ_КЛЮЧ_ЗДЕСЬ';
   ```

## Шаг 5: Готово! 🎉

Теперь Mirtube может искать настоящие видео с YouTube!

### Что теперь работает:

✅ Поиск реальных видео с YouTube
✅ Просмотр видео через встроенный YouTube плеер
✅ Реальные данные: просмотры, лайки, описание
✅ Миниатюры и аватары каналов с YouTube
✅ Если API не работает - используются локальные видео

### Примеры поиска:

- "игры"
- "музыка"
- "приколы"
- "обзоры"
- любой запрос на русском или английском!

---

**Важно:** YouTube API имеет лимит 10,000 запросов в день (бесплатно). Этого хватит для тестирования!
