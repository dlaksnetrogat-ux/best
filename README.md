# 🎬 Mirtube - YouTube Clone

Полнофункциональный клон YouTube с интеграцией YouTube API!

## ✨ Возможности

✅ Поиск настоящих видео с YouTube
✅ Встроенный YouTube плеер
✅ Shorts с вертикальной прокруткой
✅ Комментарии и лайки
✅ Темная тема как в YouTube
✅ Адаптивный дизайн

## 🚀 Быстрый старт

### Вариант 1: Автоматический запуск (Windows)
```bash
# Просто дважды кликни на файл:
start-server.bat
```

### Вариант 2: Ручной запуск

**Python:**
```bash
python -m http.server 8000
```

**Node.js:**
```bash
npx http-server -p 8000
```

Затем открой: **http://localhost:8000**

## 🔑 Настройка YouTube API (опционально)

Без API ключа работает локальный поиск по встроенным видео.

С API ключом можешь искать миллионы видео с YouTube!

1. Получи бесплатный API ключ: [Инструкция](YOUTUBE_API_SETUP.md)
2. Открой `script.js`
3. Замени строку:
   ```javascript
   const YOUTUBE_API_KEY = 'AIzaSyDemoKeyForTesting123456789';
   ```
   На свой ключ:
   ```javascript
   const YOUTUBE_API_KEY = 'ТВОЙ_КЛЮЧ_ЗДЕСЬ';
   ```

## ⚠️ Важные замечания

1. **Локальный сервер обязателен** для YouTube видео
2. YouTube API имеет лимит 10,000 запросов/день (бесплатно)
3. Без API ключа работает только локальный поиск

## 🐛 Решение проблем

### Ошибка 153 (YouTube)
**Причина:** Файл открыт через `file://`
**Решение:** Запусти локальный сервер (см. выше)

### Видео не находятся
**Причина:** Нет API ключа или неверный ключ
**Решение:** Проверь API ключ в `script.js`

### API ошибка 403
**Причина:** API ключ не активирован или превышен лимит
**Решение:** Проверь настройки в Google Cloud Console
